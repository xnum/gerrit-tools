# Code of Conduct

## Our Standards

- Be respectful and constructive.
- Focus feedback on technical content.
- Assume good intent and communicate clearly.
- Harassment and personal attacks are not tolerated.

## Enforcement

Project maintainers may remove or reject comments, issues, or contributions that violate this code of conduct.

## Scope

This applies to project spaces, including issues, pull requests, and code review discussions.

## Reporting

Report incidents to project maintainers privately.
